Set up datasets in AI Catalog¶

You can use a variety of databases:

Here we will use an S3 bucket.

Save datesets¶

In [7]:
datasets = dict()
datasets['primary'] = primary_dataset
datasets['secondary'] = secondary_dataset
In [8]:
filename = 'datasets.pickle'
outfile = open(filename,'wb')
pickle.dump(datasets,outfile)
outfile.close()
